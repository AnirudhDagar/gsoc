--- layout: home search_exclude: true --- ![]({{site.baseurl}}/images/GSoC-logo-homepage.svg) This blog serves to document my work during GSoC 2020 on **TMVA PyTorch Interface** at **CERN**. The **[Toolkit for Multivariate Analysis (TMVA)](https://root.cern/manual/tmva/)** is a **machine learning** environment integrated into the **[ROOT](https://root.cern.ch/)** scientific software framework, which has been developed specifically for **high-energy physics** applications. TMVA provides functionality for building **deep neural networks** including fully connected, convolutional and recurrent layers. In addition, TMVA provides interfaces to other deep learning frameworks (scikit-learn, Keras). This project aims to extend these interfaces to one of the most popular, fast and flexible machine learning frameworks, **[PyTorch](https://pytorch.org/)**. The focus will be on the development of a production-ready TMVA interface to PyTorch, while strictly adhering to my **[proposal](https://drive.google.com/open?id=1b54yrngOjth9wUiS7iAg_rEm7J2J749d)**. ![]({{site.baseurl}}/images/CERN-HSF-GSoC-logo.png) [CERN](https://home.cern/) and [HSF](https://hepsoftwarefoundation.org/) # Posts
